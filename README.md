# Vexzy Dumper

Full Deobfuscator Bot for Roblox Lua Scripts

## Features
- Support WeAreDevs, Moonsec V3, IronBrew, Luarmor
- Auto detection obfuscator type
- Script stats analyzer
- File upload & URL support
- Cooldown system
- Owner commands

## Commands
- `.l <file/url/code>` - Deobfuscate script
- `.l stats <code/url/file>` - Show script statistics
- `.l info` - Bot info
- `.l help` - Show help

## Setup
1. `npm install`
2. Create `.env` with `DISCORD_TOKEN=your_token`
3. `npm start`

## Supported Obfuscators
- WeAreDevs
- Moonsec V3
- IronBrew
- Luarmor
- Luraph (basic)
- Veil (basic)
